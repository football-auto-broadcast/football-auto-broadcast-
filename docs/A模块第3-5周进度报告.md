### 1.3 异常日志样例

**相机初始化失败（序列号不匹配）**：
```
[WARN] Camera F92514845 not found
[ERROR] Camera F92514845 initialization failed
```
→ 解决：确认 MVS 客户端中相机序列号，修改 `configs/ingest_streaming/config.json`

**相机掉线重连**：
```
[INFO] Camera F92514845 is offline, attempting reconnect...
[INFO] Camera F92514845 reconnecting...
[SUCCESS] Camera F92514845 initialized
[SUCCESS] Camera F92514845 started grabbing
[SUCCESS] Camera F92514845 reconnected
[INFO] Camera 0 reconnected, restarting streamer...
[INFO] Created new streamer for camera 0 -> rtsp://127.0.0.1:8554/main
```

**帧尺寸不匹配（跳过该帧）**：
```
[WARN] GstRtspStreamer rtsp://127.0.0.1:8554/main frame size mismatch: got 5038848 bytes, expected 15116544 bytes (RGB24 2592x1944)
```
→ 原因：Bayer8 数据帧被跳过，软件解码流程正常处理

**RTSP 推流连接失败（MediaMTX 未启动）**：
```
[ERROR] GstRtspStreamer rtsp://127.0.0.1:8554/main pipeline error: Could not connect to server
```
→ 解决：先启动 MediaMTX 双实例，再启动采集模块
