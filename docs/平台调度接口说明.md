# 平台调度接口说明 v1

## 文档信息
- 版本：v1
- 依据：`Team Interface Contract v1.2 Frozen` §4（端口约定）、§8（模块接口契约）、§9（状态机与错误码）
- 适用模块：platform-orchestration-module（模块 E）
- 可执行文件：`platform_orchestration_service.exe`（合同 §4.1 冻结）
- HTTP 端口：`8080`（合同 §4.2 冻结）
- 协议：HTTP REST + JSON / 本机 `127.0.0.1`（合同 §4.4 冻结）
- 状态机：`idle → running`（合同 §9.1 冻结）
- 冻结要求：接口路径、请求/响应结构冻结后不得大改

---

## 1. 通用规范

### 1.1 通用响应格式

所有接口统一返回：
```json
{
  "code": 0,
  "message": "ok",
  "data": {}
}
```

### 1.2 HTTP 状态码

| HTTP Code | 含义 |
|------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 404 | 资源不存在 |
| 409 | 状态冲突（如：比赛已在运行中） |
| 500 | 内部错误 |

### 1.3 字段命名

统一使用 `snake_case`。

## 2. 比赛管理接口

### 2.1 创建比赛

**方法**：`POST`
**路径**：`/api/v1/matches`

请求体：
```json
{
  "match_id": "match_20260405_001",
  "match_name": "校队训练赛 2026-04-05",
  "match_type": "training",
  "cameras": ["cam_01", "cam_02"],
  "scheduled_duration_min": 90
}
```

响应：
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "match_id": "match_20260405_001",
    "status": "created",
    "created_at_timestamp_ms": 1712323200000
  }
}
```

### 2.2 开始比赛

**方法**：`POST`
**路径**：`/api/v1/matches/{match_id}/start`

说明：启动采集、录像、视觉分析三个模块。

请求体：无
响应：
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "match_id": "match_20260405_001",
    "status": "running",
    "started_at_timestamp_ms": 1712323205000
  }
}
```

### 2.3 结束比赛

**方法**：`POST`
**路径**：`/api/v1/matches/{match_id}/stop`

说明：停止采集、录像、视觉分析三个模块。

响应：
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "match_id": "match_20260405_001",
    "status": "completed",
    "record_duration_sec": 5400,
    "ended_at_timestamp_ms": 1712328605000
  }
}
```

### 2.4 查询比赛

**方法**：`GET`
**路径**：`/api/v1/matches/{match_id}`

响应：
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "match_id": "match_20260405_001",
    "match_name": "校队训练赛 2026-04-05",
    "status": "running",
    "created_at_timestamp_ms": 1712323200000,
    "started_at_timestamp_ms": 1712323205000,
    "record_duration_sec": 1200
  }
}
```

### 2.5 比赛列表

**方法**：`GET`
**路径**：`/api/v1/matches`

响应：
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "matches": [
      {
        "match_id": "match_20260405_001",
        "match_name": "校队训练赛 2026-04-05",
        "status": "completed"
      }
    ]
  }
}
```

**路径**：`/api/v1/ingest/matches/init`

请求体：
```json
{
  "match_id": "match_20260405_001",
  "cameras": [
    {
      "camera_id": "cam_01",
      "role": "main",
      "model": "MV-CE050-30GC",
      "lens": "6mm_C_mount",
      "stream_uri": "rtsp://127.0.0.1:8554/main"
    },
    {
      "camera_id": "cam_02",
      "role": "aux",
      "model": "MV-CE050-30GC",
      "lens": "6mm_C_mount",
      "stream_uri": "rtsp://127.0.0.1:8555/aux"
    }
  ],
  "network_config": {
    "mode": "static_ip",
    "subnet": "192.168.10.0/24"
  },
  "capture_config": {
    "internal_source_resolution": "5mp_native",
    "rtsp_output_resolution": "1920x1080",
    "fps": 25,
    "pixel_format": "bgr8_or_nv12",
    "video_codec": "h264"

  "camera_param_strategy": {
    "load_from": "mvs_user_set_or_config",
```

## 4. 状态查询接口

**方法**：`GET`
**路径**：`/api/v1/status`

响应（合同 §3.5 E 模块强制要求：状态页必须展示以下 8 项）：
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "cam_01_status": "running",
    "cam_02_status": "running",
    "current_program_camera_id": "cam_01",
    "program_output_status": "active",
    "cam_01_fps": 25.0,
    "cam_02_fps": 25.0,
    "record_duration_sec": 5400,
    "last_error": null,
    "disk_free_gb": 250,
    "modules": {
      "ingest_streaming": { "status": "running", "port": 8081 },
      "record_program": { "status": "recording", "port": 8082 },
      "vision_event": { "status": "running", "port": 8083 },
      "highlight_generation": { "status": "idle", "port": 8084 },
      "platform_orchestration": { "status": "running", "port": 8080 }
    }
  }
}
```

### 4.2 采集模块状态

**方法**：`GET`
**路径**：`/api/v1/ingest/status`

响应（合同 §9.1 A 模块状态机：`idle → initializing → running → degraded / stopped / failed`）：
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "status": "running",
    "cam_01": {
      "camera_id": "cam_01",
      "status": "running",
      "resolution": "2592x1944",
      "fps": 25.0,
      "online": true
    },
    "cam_02": {
      "camera_id": "cam_02",
      "status": "running",
      "resolution": "2592x1944",
      "fps": 25.0,
      "online": true
    }
  }
}
```

### 4.3 录像模块状态

**方法**：`GET`
**路径**：`/api/v1/record/status`

```json
{

  "data": {
    "match_id": "match_20260405_001",
    "status": "recording",
    "cam_01_cut_status": "active",

    "record_duration_sec": 5400,
    "disk_free_gb": 250,
```

## 6. 页面路由

| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 比赛控制页 | 创建/开始/结束比赛 |
| `/status` | 服务状态页 | 各模块状态、相机状态、录制进度 |
| `/preview` | 主画面预览 | 嵌入 RTSP 播放器 |
| `/matches/{match_id}` | 比赛详情页 | 比赛信息 + 集锦结果下载 |

## 7. 错误码参考

| code | 含义 |
|------|------|
| `0` | 成功 |
| `1001` | 参数错误 |
| `1002` | 资源未初始化 |
| `1003` | 输入源不可用 |
| `1004` | 视频流异常 |
| `1005` | 文件写入失败 |
| `1006` | 文件读取失败 |
| `1007` | 任务执行失败 |
| `1008` | 上游服务不可达 |
| `1009` | 资源不存在 |
| `1010` | 状态冲突 |
| `1011` | 配置错误 |
| `1012` | 工业相机 SDK 初始化失败 |
| `1013` | 工业相机枚举失败 |
| `1014` | 相机取流失败 |
| `1019` | 磁盘空间不足 |
