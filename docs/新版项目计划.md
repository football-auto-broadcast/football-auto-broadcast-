## 1. 文档信息
**项目名称**：足球赛事自动转播与赛后精彩集锦系统  
**项目周期**：7 周  
**适用架构**：Windows 原生单主机集中式架构  
**主机配置**：RTX 4080  
**计划对象**：

1.  采集与编码分发模块 
2.  录像与主画面生成模块 
3.  视觉分析模块 
4.  事件与集锦生成模块 
5.  平台与调度模块 

---

## 2. 项目总目标
在第 7 周结束前，完成一套基于 **Windows 单主机集中式架构** 的可落地系统，实现：

1.  双路 GigE 工业相机稳定接入 
2.  双路原始录像 
3.  一路自动转播主画面生成与预览 
4.  关注区域输出 
5.  进球与精彩片段候选事件输出 
6.  赛后全场精彩集锦生成 
7.  平台统一管理、状态查看、结果下载 

总体验收目标如下：

1.  两路工业相机稳定接入 
2.  系统可完成两路原始录像 
3.  系统可生成一路自动转播主画面 
4.  视觉模块可输出关注区域与候选事件 
5.  赛后可生成全场精彩集锦 
6.  平台可完成比赛管理、状态查看、预览与集锦下载 
7.  全系统可在 Windows 原生单主机架构下稳定运行
