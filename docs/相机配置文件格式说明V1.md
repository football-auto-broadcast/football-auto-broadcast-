# 相机配置文件格式说明 v1
**文档版本**：v1.0  
**适用模块**：ingest-streaming-module（A模块）  
**冻结状态**：团队统一冻结，禁止私自修改字段
---

## 1. 配置文件路径
configs/ingest-streaming-module/config.yaml / camera_config.json

## 2. 根结构
```json
{
  "data_root": "D:\\football\\data",
  "cameras": []
}

## 3.单相机配置结构（冻结字段）
{
  "camera_id": "cam_01",
  "role": "main",
  "serial_number": "从枚举工具获取的相机序列号",
  "static_ip": "192.168.10.101",
  "stream_uri": "rtsp://127.0.0.1:8554/main",
  "params": {
    "width": 1920,
    "height": 1080,
    "fps": 25,
    "exposure_time": 10000,
    "gain": 1.0
  }
}
